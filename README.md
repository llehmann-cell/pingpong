# PinPang Strava

Prototype responsive d'une application type Strava pour le tennis de table.

## Lancer

Ouvrir `index.html` directement dans le navigateur.

Pour tester l'installation PWA Android/iOS et le mode offline, servir le dossier en HTTP:

```bash
python3 -m http.server 8080
```

Puis ouvrir `http://localhost:8080`.
